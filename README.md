## 安装 Install

Execute the following command and modify `theme` in `_config.yml` to `pithiness`.

```
git clone https://github.com/focuslau/hexo_theme_myself.git
```

## 更新 Update

Execute the following command to update pithiness.

```
cd themes/pithiness
git pull
```
